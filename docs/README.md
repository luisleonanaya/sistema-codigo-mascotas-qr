# Documentación del proyecto

Esta carpeta contiene los documentos principales del sistema web para la identificación y recuperación de mascotas mediante códigos QR y geolocalización.

## Archivos

- `administracion_proyecto.pdf`: informe principal del proyecto, incluyendo fases de inicio, planeación, ejecución, control, pruebas, anexos y evidencias.
- `manual_usuario_sistema_mascotas.pdf`: manual de usuario con instrucciones y capturas de pantalla para operar el sistema.

Los documentos corresponden a una versión beta funcional del prototipo.
